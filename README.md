# ANS - BolBol App & Company Website

This repository hosts the static company website and Google Play Console compliance documents for the **BolBol** kids learning mobile application.

## 🌐 Live Pages
- **Home Page**: [https://nayansrivastav.github.io/ANS/](https://nayansrivastav.github.io/ANS/)
- **Privacy Policy**: [https://nayansrivastav.github.io/ANS/privacy.html](https://nayansrivastav.github.io/ANS/privacy.html)
- **Terms of Service**: [https://nayansrivastav.github.io/ANS/terms.html](https://nayansrivastav.github.io/ANS/terms.html)

## 📁 Repository Contents
- `index.html`: The interactive app landing page showing features, modules, bilingual storybook, and safety parameters.
- `styles.css`: Custom CSS containing Outfit/Quicksand fonts, glassmorphism, responsive grids, and animations.
- `privacy.html`: COPPA-compliant privacy policy required for Android app distribution.
- `terms.html`: Terms of service agreement.
- `assets/`: Copied app assets (app icon and module illustrations).
